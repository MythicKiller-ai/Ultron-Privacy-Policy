Ultron® Privacy Policy

1. Data Collected

Ultron® collects minimal data required to operate, such as Discord user IDs, server IDs, and message content for moderation or bot features.

2. How Data is Used

  • To provide bot features and commands.
  • To log moderation actions if applicable.
  • To prevent abuse or spam.

3. Data Storage

All data is stored securely and only for the purpose of bot functionality. No data is sold or shared with third parties.

4. User Rights

  • You can request deletion of any stored data related to your user ID.
  • You have the right to know what data is collected.

5. Contact
