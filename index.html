Ultron™ Privacy Policy

Effective Date: 18 December 2025

1. Introduction

Ultron™ values your privacy. This Privacy Policy explains how we collect, use, and protect information when you interact with the Ultron™ Discord bot.

2. Data Collection

Ultron™ may collect the following information to provide its services:

  • Discord user ID and username for commands and moderation
  • Server ID and channel information to manage settings and logs
  • Messages content for features like AFK, economy, giveaways, and logging (if enabled)
  • Bot usage statistics for analytics and improvements

3. Data Usage

The data collected is used for:

  • Providing bot commands and features
  • Server moderation and logging
  • Tracking user activity for economy, games, and statistics
  • Improving the bot’s performance and features

4. Data Sharing

Ultron™ does not sell or share your personal information with third parties outside the scope of bot functionality. Some limited data may be shared with Discord servers if required for commands or features (like NQN or mass unban logs).

5. Data Security

We implement reasonable measures to protect your information. However, we cannot guarantee absolute security. Users are responsible for server permissions and data shared within Discord servers.

6. User Control

You can control the data processed by Ultron™:

  • Opt out of certain bot features like logging or economy commands
  • Request deletion of your server data by contacting the bot developer

7. Updates to This Policy

Ultron™ may update this Privacy Policy periodically. Continued use of the bot indicates acceptance of the updated terms. Users should check this page for the latest version.

8. Contact

If you have questions regarding this Privacy Policy, please contact the bot developer through Discord or the official support server.
